HTML & CSS quick-reference guide

HTML Guide

Getting started with HTML

HTML stands for Hyper Text Markup Language
HTML is the standard markup language for creating Web pages
HTML describes the structure of a Web page
HTML consists of a series of elements
HTML elements tell the browser how to display the content

Example code:- 

                            <!DOCTYPE html> 
                            <html> 
                            <head> 
                            <title>Page Title</title>
                            </head>
                            <body>
                            
                            </body>
                            </html>

Code explanation:-
The <!DOCTYPE html> declaration defines that this document is an HTML5 document
The <html> element is the root element of an HTML page
The <head> element contains meta information about the HTML page
The <title> element specifies a title for the HTML page (which is shown in the browser's title bar or in the page's tab)
The <body> element defines the document's body, and is a container for all the visible contents, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.

HTML elements

An HTML element is defined by a start tag, some content, and an end tag. Example code:- 

                            <body>
                            <h1>Hello World! </h1>
                            <p>Computer </p>
                            </body>

Code explanation:-
The element defines the document's body.It has a start tag <body> and an end tag </body>.
Then, inside the element there are two other elements: <h1> and <p>
The <h1> element defines a heading. It has a start tag <h1> and an end tag </h1>
The <p> element defines a paragraph. It has a start tag <p> and an end tag </p>

Atrributes

HTML Atributes

  • All HTML elements can have attributes
  • Attributes provide additional information about elements
  • Attributes are always specified in the start tag
  • Attributes usually come in name/value pairs like: name="value"

src Atrribute


The tag is used to embed an image in an HTML page. The src attribute specifies the path to the image to be displayed
Example code:- 
                            <>img src="img.jpg"<>

Code explanation:-
There are two ways to specify the URL in the src attribute:
1. Absolute URL - Links to an external image that is hosted on another website.
2. Relative URL - Links to an image that is hosted within the website. Here, the URL does not include the domain name.

href Atrribute

The tag defines a hyperlink. The href attribute specifies the URL of the page the link goes to Example code:- 
                                <a href="https://www.google.com/">Visit Google</a>

Code explanation:-
The most important attribute of the element is the href attribute, which indicates the link's destination.The link text is the part that will be visible to the reader. Clicking on the link text, will send the reader to the specified URL address.

width and height Attribute

Example code:- 
                            <img src="image.jpg" width="400" height="400">

Code explanation:-
The <img> tag should also contain the width and height attributes, which specify the width and height of the image (in pixels)

alt Atribute

Example code:- 
                            <img src="image.jpg" alt="img1">

Code explanation:-
The required alt attribute for the tag specifies an alternate text for an image, if the image for some reason cannot be displayed. This can be due to a slow connection, or an error in the src attribute, or if the user uses a screen reader.

Style Atribute

Example code:- 
                            <p style="color:rgb(255, 0, 225);"> Hello World!</p>

Code explanation:-
The style attribute is used to add styles to an element, such as color, font, size, and more.

Tags

Heading tag

HTML headings are titles or subtitles that you want to display on a webpage.
Example code:- 

                            <h1>Heading 1</h1>
                            <h2>Heading 2</h2>
                            <h3>Heading 3</h3>
                            <h4>Heading 4</h4>
                            <h5>Heading </h5>
                            <h6>Heading 6</h6>

Code explanation:-
HTML headings are defined with the <h1> to <h6> tags. <h1> defines the most important heading. <h6> defines the least important heading.

Paragraph tag

A paragraph always starts on a new line, and is usually a block of text.
Eample code:- 
                                <p>What a beautiful day.</p>

Code explanation:-
The HTML

element defines a paragraph. A paragraph always starts on a new line, and browsers automatically add some white space (a margin) before and after a paragraph.

Line break tag

The <br> tag is an empty tag, which means that it has no end tag
Example code:- 
                                <p>Apple a day <br> keeps doctor away.</p>

Code explanation:-
The HTML <br> element defines a line break. Use <br> if you want a line break (a new line) without starting a new paragraph

div tag

The <div> element is used as a container for other HTML elements.
Example code:- 
                                <div>
                                    <p>Pink flower.</p>
                                </div>

Classes

The HTML class attribute is used to specify a class for an HTML element. HTML elements can share the same class.
Example Code 

                        <div class="days">
                            <h2>Days</h2>
                            <p>Sunday </p>
                            <p>Monday</p>
                            <p>Tuesday</p>
                            <p>Wednesday</p>
                            </p>Thursday</p>
                            </p>Friday</p>
                            </p>Saturday</p>
                        </div>

Code explanation:-
The class attribute is often used to point to a class name in a style sheet. It can also be used by a JavaScript to access and manipulate elements with the specific class name.In the following example we have one <div> elements with a class attribute with the value of "days".

id

The HTML id attribute is used to specify a unique id for an HTML element.You cannot have more than one element with the same id in an HTML document.
Example code:- 

                        </h1 id="name" >Sofia</h1>

Code explanation:-
In the following example we have an <h1> element that points to the id name "name".

CSS Guide

Getting started with CSS

CSS is the language we use to style an HTML document.
CSS describes how HTML elements should be displayed.
This tutorial will teach you CSS from basic to advanced.

CSS Seclectors

A CSS selector selects the HTML element(s) you want to style.

Element selectors

The element selector selects HTML elements based on the element name.
Example code:- 
                        p {
                            text-align: center;
                            color: green;
                          }

Code explanation:-
Here, all <p> elements on the page will be center-aligned, with a green text color

class selectors

The class selector selects HTML elements with a specific class attribute.
To select elements with a specific class, write a period (.) character, followed by the class name.
Example code:- 
                        .text {
                            text-align: center;
                            color: orange;
                          }

Code explanation:-
In this example all HTML elements with class="text" will be red and center-aligned and text color as orrange

id selectors

The id selector uses the id attribute of an HTML element to select a specific element.
The id of an element is unique within a page, so the id selector is used to select one unique element!
To select an element with a specific id, write a hash (#) character, followed by the id of the element.
Example code:- 
                        #para1 {
                            text-align: left;
                            color: red;
                          }

Code example:-
The CSS rule below will be applied to the HTML element with id="para1

Universal Selector

The universal selector (*) selects all HTML elements on the page. Example code:- 
                        * {
                            text-align: center;
                            color: blue;
                          }

Code explanation:-
The CSS rule below will affect every HTML element on the page

Padding

Padding is used to create space around an element's content, inside of any defined borders
Example code:- 

                        div {
                            padding: 70px;
                            border: 1px solid #4CAF50;
                          }

Code explanation:-
Space arround div will be 70px

Margin

Margins are used to create space around elements, outside of any defined borders.
Eample code:- 

                        p{
                            margin: 40px;
                        }

Code explanation:-
Set margins for all four sides of a <p> element

The display Property

The display property is used to specify how an element is shown on a web page. Every HTML element has a default display value, depending on what type of element it is. The default display value for most elements is block or inline. The display property is used to change the default display behavior of HTML elements.

Block-level Elements

A block-level element ALWAYS starts on a new line and takes up the full width available (stretches out to the left and right as far as it can).

Inline Elements

An inline element DOES NOT start on a new line and only takes up as much width as necessary.

Position Property

The position property specifies the type of positioning method used for an element (static, relative, fixed, absolute or sticky).

Static Position

HTML elements are positioned static by default.
Static positioned elements are not affected by the top, bottom, left, and right properties.
An element with position: static; is not positioned in any special way; it is always positioned according to the normal flow of the page
Example code:- 
                        .position {
                            position: relative;
                            padding-left: 30px;
                          }

Fixed Position

An element with position: fixed; is positioned relative to the viewport, which means it always stays in the same place even if the page is scrolled. The top, right, bottom, and left properties are used to position the element.
A fixed element does not leave a gap in the page where it would normally have been located.
Example code:- 
                        .position {
                            position: fixed;
                            margin:30px;
                            border: 3px solid #73AD21;
                          }

Absolute Position

An element with position: absolute; is positioned relative to the nearest positioned ancestor (instead of positioned relative to the viewport, like fixed). However; if an absolute positioned element has no positioned ancestors, it uses the document body, and moves along with page scrolling.Example code:- 
                        .position{
                            position: absolute;
                            width: 200px;
                            height: 100px;
                        }

Sticky position:-

An element with position: sticky; is positioned based on the user's scroll position. A sticky element toggles between relative and fixed, depending on the scroll position. It is positioned relative until a given offset position is met in the viewport - then it "sticks" in place (like position:fixed).
Exammple code:- 
                        .position {
                            position: sticky;
                            background-color: red;
                            border: 2px solid #4CAF50;
                          }
